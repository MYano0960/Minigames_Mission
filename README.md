# Minigames_Mission
和歌山県の魅力を伝えるためのミニゲーム集。ミニゲームは随時追加予定です。


## デモ
https://drive.google.com/file/d/100PhjnyvACmkk24pdi0DIHAx2rb55P2j/view?usp=sharing


## インストール方法
１．リポジトリをクローン
２．事前にUnityをインストールする。
3.Unity Editorからファイルを実行する。


## 操作方法
基本的にクリックのみで動作させることが可能である。


## ファイル構成
Minigames_Missions
┣  Assets
┃ ┣  Resources # ゲームを構成するための画像の素材データ
┃ ┣  Scenes # ゲームのシーンデータ
┃ ┣  Scripts # C#スクリプトデータ
┃ ┣  Settings
┃ ┗  TextMesh Pro
┣ Packages
┣ ProjectSettings
┣ .gitattributes
┣ .gitignore
┗ README.md
